# Data Analysis and Visualization System using Python

## Project Overview

The Data Analysis and Visualization System is a Python-based project that analyzes sales data from a CSV file and generates meaningful business insights through statistical analysis and visualizations.

The project uses Pandas for data processing, Matplotlib and Seaborn for visualization, and performs data cleaning before analysis.

---

## Features

- Load sales data from a CSV file
- Detect missing values
- Remove missing and duplicate records
- Calculate total sales and total profit
- Identify the highest-selling product
- Generate category-wise sales analysis
- Produce summary statistics
- Create sales and profit visualizations
- Generate top 5 products analysis
- Save charts automatically as image files

## Data Cleaning

The project performs the following preprocessing steps:

- Checks for missing values
- Removes missing records using `dropna()`
- Removes duplicate records using `drop_duplicates()`
